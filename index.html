Welcome to SR-CTF

The project SR-CTF is being sponsored by the NSF with the goal of enhancing and broadening computer security education through the development of Stepwise and Reusable CTF challenges that emphasize problem solving.

Stepwise

A user's skillset should grow as they progress from one challenge to the next. That growth should be relied upon for later challenges. A natural progression from challenge to challenge should be evident.

Reusable

Each challenge should be tunable such that unique variants can be generated with minimal effort.

Get Started

The project consists of two main part:

  • A website framework to be installed on server, the website is used to hold all challenge contents.
  • Challenge sequences with different security topics

Set up Website Framework

  • First get this repository, follow the instructions to set up your server

Imported Challenge Sequences

  • Follow the instructions and import challenge contents into the server, then you are all set!

  • Available challenge sequences:

    • Binary Exploitation(Pwn)

    • Web Exploitation(Web)
